Layla

I make websites and web applications in Växjö, Sweden. I believe that software can solve many of the world's ills and make the world a better place.

I'm very interested in

Gamification. Can we make a program for doing your taxes that plays like a fun game?

Simplicity. Can we make software that "just works" so that users don't even notice they're using it?

Communication. How much communication can we cram into the shortest time span available? Is it true that a single picture can communicate 1000 words in just a glance?

When I'm not programming, I enjoy sci-fi, Korean movies, swing music and dance and running up the down escalator.
