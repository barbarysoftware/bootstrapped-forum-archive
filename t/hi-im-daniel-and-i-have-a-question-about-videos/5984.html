If you just did voice over screen captures a tool like Camtasia costs about $300/yr (See https://www.techsmith.com/store/camtasia ) and video hosting on Vimeo (http://www.vimeo.com) will only run you another $300/yr. A site like https://www.123rf.com/ has stock sound (there are many others) so I think you could get start much more cheaply than you have calculated. The kind of technical content you are talking about developing should be a good fit for a voice over screen capture approach. We have done several demo videos for BeamWise using this approach
